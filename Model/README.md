# 3Dモンテカルロシミュレーション (`Model/`)

このフォルダには、水星のナトリウム大気を3Dモンテカルロ法でシミュレーションするためのコアコードが格納されています。

---

## 概要
このシミュレーションは、光刺激脱離、熱脱離といった複数の物理プロセスを考慮し、水星周辺のナトリウム粒子の振る舞いを追跡します。
最終的なアウトプットとして、3Dの数密度グリッド（`.npy` ファイル）を生成します。

---

## ファイル構成
* `mkNacolumnDensity8.1.py`
    * シミュレーションを実行するメインスクリプトです。複数の物理プロセスを統合したものです。
* `mkNacolumDensity7.1_〇〇.py` 
    * 個別の物理プロセスでの粒子のシミュレーションを行うコードです。
* `mkOrbitDataMercury.pu` 
    * シミュレーションで使うパラメータを計算するファイルです。
* `Visualize_NaColumnDensity.py`
    * `analysis_demo.ipynb` の元になった、3D密度グリッドをプロットするためのスクリプトです。

---

## 実行方法
シミュレーションを実行するには、`mkNacolumnDensity8.1.py` を呼び出します。
main関数に定数などの設定があるため、適宜設定を行います。

なお、mkNaColumnDenssity8.1.pyは現段階では未完成です。
