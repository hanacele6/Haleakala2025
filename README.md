# 水星ナトリウム大気 解析コード (Haleakala2025)



## 概要
このリポジトリには、水星のナトリウム大気の特性を分析するためのPythonコードが格納されています。
主に、観測生データの処理と、3Dモンテカルロシミュレーションの2つの機能で構成されています。

* **`2025ver` (データ前処理):**
    ハワイの望遠鏡で観測された生データ（Raw Data）を、研究で使用可能な形式に整形（前処理）するためのパイプラインです。
* **`Model` (シミュレーション):**
    水星の大気を3Dモンテカルロ法でシミュレーションし、その振る舞いを再現するためのコードです。

---

## 使用環境
* **言語:** Python 3.9.21
* **主要ライブラリ:** NumPy, Matplotlib, Astropy
* **ツール:** Jupyter Notebook, VS Code

---

## プロジェクトの背景
水星のナトリウム大気の生成メカニズムは、未だ完全には解明されていません。
本研究では「季節変動性」や「朝夕非対称性」がなぜ生じるのかを詳細に分析し、その観測パターンを各生成メカニ(中略)...水星大気の起源という大きな謎に迫ることを主要な目的としています。
