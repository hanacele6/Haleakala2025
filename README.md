# Haleakala2025



## 概要
ここにあるコード群は、水星の大気の特性を調べるための解析コードです。

OLDverに入っているコードは先行研究で使用されていたIDLのコードをPythonへ移植することを試みたコードです。現在は使用しておりません。
2025verに入っているコードはハワイにある望遠鏡で得られた生データを解析で使用できる形にするためのものです。
Modelに入っているコードは水星の大気をモンテカルロシミュレーションを用いて再現することを目的にしたものです。

詳しくはそれぞれのファイルに入っているReadMeを参照してください。

##　プロジェクトの背景
水星のナトリウム大気の生成メカニズムは、未だ完全には解明されていない。
そこで、本研究では「季節変動性」や「朝夕非対称性」がなぜ生じるのかを詳細に分析し、その観測パターンを各生成メカニズムの理論モデルと比較することで、水星大気の起源という大きな謎に迫ることを主要な目的としている。
